# 第一章学习笔记 - 初识智能体

## 📌 核心概念

> 智能体是能够通过传感器感知环境，并自主地通过执行器采取行动以达成目标的实体。

## 🧠 关键知识点

### 概念 1：传统智能体的分类
- **简单反射智能体**：基于"条件-动作"规则，无记忆（如自动恒温器）
- **基于模型的反射智能体**：拥有内部世界模型，能追踪环境状态
- **基于目标的智能体**：主动规划，选择导向目标的行动（如 GPS 导航）
- **基于效用的智能体**：最大化期望效用，在多目标间权衡
- **学习型智能体**：通过强化学习自我改进（如 AlphaGo）

### 概念 2：LLM 驱动的新范式
- 核心引擎从规则变为大语言模型
- 知识来源从人工编码变为预训练数据
- 交互方式从结构化输入变为自然语言

### 概念 3：智能体循环（Agent Loop）
1. **感知（Perception）**：接收环境输入
2. **思考（Thought）**：规划 + 工具选择
3. **行动（Action）**：执行具体操作
4. **观察（Observation）**：获取行动结果反馈

## 📝 学习总结

1. 智能体的核心在于"感知-思考-行动"的闭环。
2. LLM 让智能体从"规则驱动"变为"数据驱动"，更灵活但也更难预测。
3. 智能体循环是理解所有智能体框架的基础。
