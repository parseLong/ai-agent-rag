# Claude Code 设置与配置

## 配置层级（从高到低）

| 优先级 | 位置 | 范围 | 共享? |
|:---:|------|------|:---:|
| 1 | Managed settings | 组织 | 是 |
| 2 | 命令行参数 | 会话 | - |
| 3 | `.claude/settings.local.json` | 项目 | 否（git-ignored） |
| 4 | `.claude/settings.json` | 项目 | 是 |
| 5 | `~/.claude/settings.json` | 用户 | - |

## 核心配置

### 通用设置

| 键 | 类型 | 默认值 | 说明 |
|---|------|------|------|
| `model` | string | `"default"` | 覆盖默认模型 |
| `agent` | string | - | 设置主对话的默认代理 |
| `language` | string | `"english"` | Claude 的首选响应语言 |
| `cleanupPeriodDays` | number | `30` | 清理间隔（天） |
| `autoUpdatesChannel` | string | `"latest"` | 发布频道：`"stable"` 或 `"latest"` |
| `alwaysThinkingEnabled` | boolean | `false` | 默认启用扩展思考 |
| `defaultShell` | string | `"bash"` | 默认 shell（`bash` 或 `powershell`） |
| `tui` | string | `"default"` | 渲染模式：`"fullscreen"` 或 `"default"` |
| `includeGitInstructions` | boolean | `true` | 包含内置 git 指令 |

### 模型别名

| 别名 | 说明 |
|------|------|
| `"default"` | 根据账户类型推荐 |
| `"sonnet"` | Claude Sonnet 4.6（最新） |
| `"opus"` | Claude Opus 4.6（最新） |
| `"haiku"` | 快速 Haiku 模型 |
| `"sonnet[1m]"` | Sonnet + 1M 上下文 |
| `"opus[1m]"` | Opus + 1M 上下文 |
| `"opusplan"` | 计划用 Opus，执行用 Sonnet |

### 努力级别 (Effort Level)

| 级别 | 说明 |
|------|------|
| `max` | 最大推理深度（Opus 4.6 专属） |
| `xhigh` | 扩展高推理深度（Opus 4.7 专属，默认） |
| `high` | 完整推理深度（Opus 4.6/Sonnet 4.6 默认） |
| `medium` | 平衡推理 |
| `low` | 最小推理，最快响应 |

## 权限设置

### 权限结构

```json
{
  "permissions": {
    "allow": [],
    "ask": [],
    "deny": [],
    "additionalDirectories": [],
    "defaultMode": "acceptEdits",
    "disableBypassPermissionsMode": "disable"
  }
}
```

### 权限模式

| 模式 | 说明 |
|------|------|
| `"default"` | 标准权限检查，带提示 |
| `"acceptEdits"` | 自动接受文件编辑 |
| `"dontAsk"` | 自动拒绝，除非已通过权限规则预批准 |
| `"bypassPermissions"` | 跳过所有权限检查（危险） |
| `"auto"` | 自动批准，带后台安全检查 |
| `"plan"` | 只读探索模式 |

### 工具权限语法

| 工具 | 语法 | 示例 |
|------|------|------|
| `Bash` | `Bash(command pattern)` | `Bash(npm run *)` |
| `Read` | `Read(path pattern)` | `Read(.env)` |
| `Edit` | `Edit(path pattern)` | `Edit(src/**)` |
| `Write` | `Write(path pattern)` | `Write(*.md)` |
| `WebFetch` | `WebFetch(domain:pattern)` | `WebFetch(domain:example.com)` |
| `WebSearch` | `WebSearch` | 全局搜索 |
| `Task` | `Task(agent-name)` | `Task(Explore)` |
| `Agent` | `Agent(name)` | `Agent(researcher)` |
| `Skill` | `Skill(skill-name)` | `Skill(weather-fetcher)` |
| `MCP` | `mcp__server__tool` | `mcp__memory__*` |

## 工作树设置 (Worktree)

```json
{
  "worktree": {
    "symlinkDirectories": ["node_modules", ".cache"],
    "sparsePaths": ["packages/my-app", "shared/utils"],
    "baseRef": "fresh"
  }
}
```

## 沙盒设置 (Sandbox)

```json
{
  "sandbox": {
    "enabled": true,
    "autoAllowBashIfSandboxed": true,
    "excludedCommands": ["git", "docker", "gh"],
    "network": {
      "allowLocalBinding": true,
      "allowedDomains": ["*.example.com"]
    },
    "filesystem": {
      "denyWrite": ["./.env"],
      "denyRead": ["./secrets/"]
    }
  }
}
```

## 状态行配置 (Status Line)

```json
{
  "statusLine": {
    "type": "command",
    "command": "git branch --show-current 2>/dev/null || echo 'no-branch'",
    "padding": 2,
    "refreshInterval": 5
  }
}
```

## 常用环境变量

| 变量 | 说明 |
|------|------|
| `ANTHROPIC_API_KEY` | API 密钥 |
| `ANTHROPIC_BASE_URL` | 自定义 API 端点 |
| `ANTHROPIC_MODEL` | 模型名称 |
| `CLAUDE_CODE_EFFORT_LEVEL` | 努力级别 |
| `CLAUDE_CODE_USE_BEDROCK` | 使用 AWS Bedrock |
| `CLAUDE_CODE_USE_VERTEX` | 使用 Google Vertex AI |
| `CLAUDE_CODE_SIMPLE` | 简化模式 |
| `DISABLE_TELEMETRY` | 禁用遥测 |
| `DISABLE_AUTOUPDATER` | 禁用自动更新 |
| `BASH_MAX_TIMEOUT_MS` | Bash 命令超时 |
| `MAX_MCP_OUTPUT_TOKENS` | MCP 最大输出令牌 |
| `MCP_TIMEOUT` | MCP 启动超时 |

## 完整示例

```json
{
  "$schema": "https://json.schemastore.org/claude-code-settings.json",
  "model": "sonnet",
  "agent": "code-reviewer",
  "language": "english",
  "cleanupPeriodDays": 30,
  "autoUpdatesChannel": "stable",
  "alwaysThinkingEnabled": true,
  "defaultShell": "bash",
  "tui": "fullscreen",
  
  "permissions": {
    "allow": [
      "Edit(*)",
      "Write(*)",
      "Bash(npm run *)",
      "Bash(git *)",
      "mcp__*",
      "Agent(*)"
    ],
    "deny": [
      "Read(.env)",
      "Read(./secrets/**)"
    ],
    "defaultMode": "acceptEdits"
  },

  "enableAllProjectMcpServers": true,

  "sandbox": {
    "enabled": true,
    "excludedCommands": ["git", "docker"],
    "filesystem": {
      "denyWrite": ["./.env"],
      "denyRead": ["./secrets/"]
    }
  },

  "statusLine": {
    "type": "command",
    "command": "git branch --show-current"
  },

  "env": {
    "NODE_ENV": "development"
  }
}
```
