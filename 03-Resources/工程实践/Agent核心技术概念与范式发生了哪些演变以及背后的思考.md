# Agent核心技术概念与范式发生了哪些演变以及背后的思考

> 阿里妹导读，作者飞樰，2026-05-22

## 核心内容

本文梳理了 Agent 从 2023 到 2026 年的技术范式演变，包含两个核心框架：

1. **四阶段发展模型**：被动式 ReAct → 工作流 Agent → 自主 Agent → 自进化 Agent
2. **六维度范式重构**：Prompt / Planning / Memory / Tools / Workflow / Environment
