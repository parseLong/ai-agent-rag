# 快速上手指南

> 欢迎来到 AI Agent 第二大脑！这份指南将帮助你在 5 分钟内开始使用。

---

## 📌 第一步：了解你的知识库

你已经拥有了一个结构化的知识管理系统，包含以下文件夹：

| 文件夹            | 用途               |
| -------------- | ---------------- |
| `00-Inbox`     | 临时笔记，快速记录想法      |
| `01-Projects`  | 项目实践，记录代码和实验     |
| `02-Areas`     | 领域笔记，按章节/主题组织    |
| `03-Resources` | 参考资料，外部文章、书籍笔记   |
| `04-Permanent` | 永久笔记，精炼的核心概念卡片   |
| `05-Archives`  | 归档内容，不再活跃但需保留的笔记 |
| `Templates`    | 模板文件，新建笔记时使用     |
| `MOCs`         | 内容地图，知识的导航入口     |
| `Assets`       | 图片、附件存储          |

---

## 📌 第二步：学习如何记笔记

### 场景 1：学完一章课程

1. 打开 `Templates/学习笔记模板.md`
2. 复制内容，在 `02-Areas/` 中新建文件
3. 按模板结构填写：核心概念、关键知识点、代码实践、疑问等
4. 添加标签，如 `#学习笔记 #第一章 #智能体`

### 场景 2：遇到重要概念

1. 打开 `Templates/概念卡片模板.md`
2. 复制内容，在 `04-Permanent/` 中新建文件
3. 用自己的话解释概念，建立知识关联
4. 添加标签，如 `#概念卡片 #永久笔记 #ReAct`

### 场景 3：做项目实践

1. 打开 `Templates/项目模板.md`
2. 复制内容，在 `01-Projects/` 中新建文件
3. 记录项目目标、实现步骤、问题与解决方案
4. 添加标签，如 `#项目 #实践 #智能旅行助手`

---

## 📌 第三步：建立知识链接

在笔记中使用 `[[双链语法]]` 创建链接：

```markdown
# 我的学习笔记

今天学习了 ReAct 范式，它是 [[智能体（Agent）]] 的一种重要实现方式。

## 关联概念
- [[ReAct 范式]]
- [[智能体循环]]
- [[工具调用]]
```

**查看链接关系**：
- 按 `Ctrl/Cmd + G` 打开图谱视图
- 查看笔记之间的关联网络
- 发现知识盲区，补充链接

## 📌 第五步：每日习惯

### 每日学习流程

```
1. 打开 Obsidian
2. 在 00-Inbox 中快速记录今日学习目标
3. 阅读教材章节
4. 创建学习笔记（使用模板）
5. 提取核心概念，创建概念卡片
6. 完成练习，记录疑问
7. 更新学习进度
```

### 每周回顾流程

```
1. 回顾本周创建的笔记
2. 整理 00-Inbox 中的临时笔记
3. 将成熟的笔记移动到 04-Permanent
4. 检查并补充知识链接
5. 更新 MOC 文件
```

---

## 💡 小贴士

1. **不要追求完美**：先完成再完美，笔记可以随时修改
2. **多用链接**：链接比标签更能建立知识网络
3. **定期回顾**：知识需要反复回顾才能内化
4. **记录疑问**：不懂的地方一定要记录下来
5. **分享输出**：尝试用自己的话写博客或做分享
