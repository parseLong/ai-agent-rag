# 智能体（Agent）

## 📖 定义

智能体是任何能够通过**传感器（Sensors）**感知其所处**环境（Environment）**，并**自主**地通过**执行器（Actuators）**采取**行动（Action）**以达成特定目标的实体。

## 🧠 我的理解

智能体 = 感知 + 决策 + 行动。

它不是简单的输入输出程序，而是一个能够根据环境变化自主调整行为的系统。就像人一样，通过眼睛看（感知）、大脑思考（决策）、手脚行动（执行）。

## 🎯 为什么重要

智能体是 AI 从"工具"走向"协作者"的关键。传统的 AI 是被动响应（你问我答），而智能体是主动解决问题（你给目标，它去执行）。

## 🔗 关联知识

- **相似概念**：[[机器人]]、[[自动化程序]]
- **相反概念**：[[被动工具]]（如传统搜索引擎）
- **前置知识**：[[传感器]]、[[执行器]]、[[环境]]
- **应用场景**：[[智能旅行助手]]、[[自动驾驶]]、[[自动客服]]

## 💡 实例/案例

**智能旅行助手**：
- **感知**：接收用户"规划厦门之旅"的指令
- **决策**：分解为查询天气、搜索景点、预订酒店等子任务
- **行动**：调用天气 API、调用搜索引擎、调用预订接口

## 📝 个人笔记

现代 LLM 驱动的智能体与传统智能体的核心区别在于：传统智能体的规则是人工编写的，而 LLM 智能体的"规则"是从海量数据中学习到的模式。这使得 LLM 智能体更灵活、更通用，但也更难预测和控制。
