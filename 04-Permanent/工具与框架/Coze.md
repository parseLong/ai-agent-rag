# Coze

## 📖 定义

Coze（扣子）是字节跳动推出的零代码/低代码 AI 智能体构建平台，主打可视化编排和一键多平台发布。它拥有极其友好的可视化界面，用户可以通过拖拽插件、配置知识库和设定工作流来创建智能体，无需编程背景即可轻松创造。

## 🧠 我的理解

Coze 的定位是"智能体的乐高积木"。它将复杂的 AI 应用开发简化为可视化的积木搭建，让非技术人员也能参与智能体的创造。其核心优势在于：极低的上手门槛、丰富的插件生态、以及强大的多平台发布能力（一键发布到微信、飞书、抖音、豆包等）。

## 🎯 为什么重要

Coze 代表了 AI 应用开发"民主化"的趋势。它极大地降低了智能体开发的技术门槛，让产品经理、运营人员甚至普通用户都能快速将创意变为可交互的 AI 应用，加速了 AI 技术的普及和应用创新。

## 🔑 核心功能模块

| 模块 | 说明 | 比喻 |
|------|------|------|
| **工作流** | 关卡通关路线图 | 定义智能体的执行流程 |
| **对话流** | NPC 对话通关 | 管理对话交互逻辑 |
| **插件** | 角色技能卡 | 扩展智能体能力 |
| **知识库** | 游戏百科全书 | 存储领域知识 |
| **卡片** | 快捷道具栏 | 美化输出展示 |
| **提示词** | 角色的移动键 | 控制智能体行为 |
| **数据库** | "云存档" | 持久化数据存储 |

## 🔑 核心优势

- **零代码体验**：可视化拖拽，无需编程
- **丰富插件生态**：内置大量预置插件，覆盖各类场景
- **多平台发布**：一键发布到微信、飞书、抖音、豆包等
- **多智能体模式**：支持单智能体对话流、多智能体协作

## ⚠️ 局限性

- **不支持 MCP**：目前不支持 Model Context Protocol，限制了与外部工具的标准化集成
- **配置复杂度**：部分插件需要 API Key，对非技术用户有门槛
- **导出限制**：免费版无法导出标准 JSON 配置文件（付费版可导出 zip 格式，但无法跨平台使用）
- **编排限制**：复杂工作流编排需要一定的 JavaScript 或 Python 基础

## 💡 实例/案例

在 Hello-Agents 教程中，使用 Coze 构建了一个"每日AI简报"智能体，它能够：
- 从多个信息源（36氪、虎嗅、GitHub、arXiv）抓取最新 AI 资讯
- 自动整合生成结构化、专业化的简报
- 支持多渠道发布

### 案例详解：每日 AI 简报

**信息源插件配置**：
- **RSS 插件**：订阅 36氪、虎嗅、IT之家、InfoQ 的 RSS 源
- **GitHub 插件**：追踪 AI 相关开源项目动态
- **arXiv 插件**：获取最新学术论文

**角色与提示词设计**：
将智能体设定为"资深科技媒体编辑"，通过系统提示词定义输出格式：
```
# 角色
你是一位资深且权威的科技媒体编辑...

## 工作流
1. 日报开头显著标注"AI日报"和当天日期
2. 根据内容在标题开头添加 Emoji
3. 所有内容必须与AI、LLM、AIGC高度相关
4. 必须为每一条目提供原始链接
5. 对每条新闻进行简短、精准的概况描述
```

**多渠道发布**：
完成测试后，可一键发布到微信、飞书、抖音、豆包等多个主流平台。
