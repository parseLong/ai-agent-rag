# 机器学习基础

## 📖 定义

机器学习是一门**多领域交叉学科**，涉及概率论、统计学、逼近论、凸分析、算法复杂度理论等多门学科。它专门研究计算机怎样模拟或实现人类的学习行为，以获取新的知识或技能，重新组织已有的知识结构使之不断改善自身的性能。

## 🧠 我的理解

机器学习的本质是"从数据中学习规律"。监督学习像学生对照着标准答案写作业，无监督学习像学生自己总结学习规律，强化学习像学生在游戏中摸索规则，通过胜败不断优化学习策略。

## 🎯 为什么重要

机器学习是[[深度学习]]和[[大语言模型（LLM）]]的上层基础。理解三大范式（监督、无监督、强化）的适用场景，才能选择正确的技术路线解决实际问题。

## 三大范式对比

| 范式 | 数据需求 | 学习方式 | 比喻 | 典型应用 |
|------|---------|---------|------|---------|
| 监督学习 | 需标注数据 | 对照答案学习 | 学生对照标准答案写作业 | 天气预报、疾病诊断、图像识别 |
| 无监督学习 | 无标注数据 | 自行发现规律 | 学生自己总结学习规律 | 聚类分析、降维、异常检测 |
| 强化学习 | 环境反馈 | 试错+奖励优化 | 学生在游戏中摸索规则 | 游戏、自动驾驶、机器人控制 |

## 监督学习

**定义**：使用标记数据进行训练，训练数据集中包含输入特征及对应的目标标签（输出），模型的训练目标是学习到从输入特征到目标标签的映射关系。

**常见算法**：
- 线性回归
- 逻辑回归
- 决策树
- 支持向量机（SVM）
- 朴素贝叶斯

> [!example] 房价预测
> 输入特征：房屋面积、房间数量、地理位置 → 输出标签：房屋实际价格 → 通过线性回归建立房价与特征的关系模型。

**应用场景**：天气预报、疾病诊断、图像识别、垃圾邮件过滤等。

## 无监督学习

**定义**：训练数据集中没有给定的目标标签或输出，主要目的是发现数据中的内在结构和模式。

**常见算法**：
- **K-Means聚类**：将客户数据根据消费行为划分为不同聚类
- **DBSCAN密度聚类**：基于密度的空间聚类
- **PCA（主成分分析）**：降维
- **SVD（奇异值分解）**：矩阵分解

> [!example] 客户分群
> K-Means可以将一组客户数据根据消费行为特征划分为不同的聚类，每个聚类代表一类具有相似行为的客户群体。

**应用场景**：数据探索、降维、异常检测，在商业分析、网络安全、生物信息学方面有重要应用。

## 强化学习

**定义**：智能体通过与环境交互，根据奖励信号学习最优行为策略，目标是找到使长期累积奖励最大化的最优策略。

**常见算法**：
- **深度Q网络（DQN）**
- **策略梯度算法**：A2C、A3C、PPO
- **DDPG（深度确定性策略梯度）**

> [!example] 机器人导航
> 机器人（智能体）在环境中不断尝试不同动作（前进、转弯），根据是否接近目标及是否遇到障碍物获得奖励或惩罚，逐渐找到最优路径。

**应用场景**：机器人控制、游戏、自动驾驶、资源管理等。

## 模型评估

**定义**：衡量训练好的模型在实际应用中的性能表现，是确保机器学习模型质量和可靠性的关键步骤。

### 分类任务评估指标

| 指标 | 说明 |
|------|------|
| 准确率（Accuracy） | 正确预测的比例 |
| 精确率（Precision） | 预测为正的样本中真正为正的比例 |
| 召回率（Recall） | 真正为正的样本中被正确预测的比例 |
| F1值 | 精确率与召回率的调和平均 |
| [[F1 分数]] | ROC曲线下面积，综合评估分类性能 |

### 回归任务评估指标

| 指标 | 说明 |
|------|------|
| MSE（均方误差） | 预测值与实际值的平均平方差 |
| MAE（平均绝对误差） | 预测值与实际值的平均绝对差 |
| R²（决定系数） | 模型解释的方差比例 |

> [!example] 评估指标的实战选择
> - **癌症诊断**：召回率更重要——漏诊比误诊更危险
> - **垃圾邮件过滤**：精确率更重要——误判正常邮件比漏判垃圾邮件更影响用户体验
> - **股票价格预测**：MSE衡量预测值与实际价格之间的平均差异程度

### 验证方法

- **留出法**：直接划分训练集/测试集
- **交叉验证法**：K折交叉验证防止过拟合
- **自助法（Bootstrap）**：通过重采样评估模型稳定性
