# 大模型核心术语

## 📖 定义

大模型核心术语是AI大模型领域的**知识地图索引**，涵盖核心技术体系、支撑技术生态、系统与架构、AI应用技术、伦理与安全技术五大板块。

## 🧠 我的理解

这些术语不是孤立的名词，而是构成一个相互关联的技术生态。算力基础设施是地基，数据工程是原料，模型架构是建筑，应用技术是功能，伦理安全是护栏。

## 核心技术体系

### 机器学习（ML）

| 子领域 | 定义 | 应用 |
|--------|------|------|
| **监督学习** | 基于标注数据训练分类/回归模型 | 垃圾邮件分类、房价预测 |
| **无监督学习** | 发现数据潜在结构 | 聚类分析、降维、对比学习 |
| **元学习** | "学会学习"，快速适应新任务 | 少样本学习 |
| **联邦学习** | 分布式隐私保护训练 | 医院联合训练模型 |
| **在线学习** | 模型实时更新适应动态数据流 | 实时推荐 |

### 深度学习（DL）

| 架构 | 特点 | 代表 |
|------|------|------|
| **[[Transformer 架构]]** | 自注意力机制驱动 | GPT系列、BERT |
| **扩散模型** | 逐步去噪生成高质量数据 | Stable Diffusion |
| **MoE** | 组合多个子模型 | Switch Transformer |
| **脉冲神经网络（SNN）** | 模拟生物神经脉冲 | — |
| **胶囊网络** | 编码空间层次关系 | — |

> 深度学习需海量数据与高性能算力支持，依赖GPU/TPU加速。

### 自然语言处理（NLP）

| 术语 | 定义 |
|------|------|
| **大语言模型（LLM）** | 千亿参数级预训练模型（如GPT-4、PaLM） |
| **思维链（CoT）** | 分步推理提升模型逻辑能力 |
| **检索增强生成（RAG）** | NLP与知识库结合，提升答案准确性 |
| **指令微调** | 通过任务指令调整模型行为 |
| **多模态对齐** | 对齐文本、图像等跨模态语义 |

### 计算机视觉（CV）

| 术语 | 定义 |
|------|------|
| **ViT** | 将Transformer应用于图像分类 |
| **NeRF** | 3D场景重建与渲染 |
| **目标跟踪** | 视频中持续追踪物体（如SiamFC） |
| **光流估计** | 计算像素级运动矢量 |
| **YOLO** | 实时目标检测 |
| **U-Net** | 医学影像分割 |
| **FaceNet** | 人脸识别特征向量比对 |

### 强化学习（RL）

| 术语 | 定义 |
|------|------|
| **PPO** | 近端策略优化，稳定策略梯度训练 |
| **模仿学习** | 从专家示范中学习策略 |
| **MARL** | 多智能体协作/竞争（如星际争霸AI） |
| **逆强化学习** | 从行为反推奖励函数 |

### 模型优化技术

| 术语 | 定义 |
|------|------|
| **知识蒸馏** | 大模型压缩为小模型（如DistilBERT） |
| **量化感知训练（QAT）** | 训练时模拟低精度计算 |
| **动态网络** | 根据输入调整模型结构（如SkipNet） |
| **稀疏训练** | 训练时自动剪枝冗余连接 |

## AGI（通用人工智能）

**定义**：具备与人类相当或超越人类的通用智能的AI系统。与弱AI（Narrow AI）不同，AGI能跨领域迁移知识、自主学习新技能。

**核心特征**：通用学习能力、抽象推理与逻辑、目标导向与自主决策、多模态感知与交互、自我意识与情感模拟。
