# 智能体（Agent）

> 使用 AI 来实现目标并代表用户完成任务的软件系统，具备推理、规划和记忆能力，具有一定的自主性。

## 核心要点

- **与传统AI的区别**：传统模型是"问答机"，你问一句它答一句；Agent是"助理"，你给它目标，它自主完成，需要查资料就查，需要调API就调
- **三大核心特征**：自主性（自己决定下一步做什么）、反应性（能根据环境变化调整）、主动性（多步行动达成目标）
- **四大核心组件**形成一个闭环：感知模块→规划模块→记忆模块→工具调用模块→回到感知

> [!tip] 面试要点
> 什么场景用Agent？需要多步骤执行、工具调用、跨系统操作的场景才需要Agent；简单的一次性问答用传统模型更轻量高效。
从 Chat 到 Agent 的范式转变，在 2023-2026 这个时间窗口内进一步细分为四个技术形态阶段：

## AI Agent 范式转变

| 阶段            | 时间   | 范式特征        | 与 Chat→Agent 转变的关系 |
| ------------- | ---- | ----------- | ------------------ |
| **被动式 ReAct** | 2023 | 增强版 Chatbot | Chat 时代的延伸         |
| **工作流 Agent** | 2024 | 工程化约束       | 从 Chat 到 Agent 的过渡 |
| **自主 Agent**  | 2025 | 长程规划+自校验    | Agent 时代的真正开启      |
| **自进化 Agent** | 2026 | 沉淀+反思+进化    | Agent 的终极形态方向      |
