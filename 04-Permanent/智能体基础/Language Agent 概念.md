# Language Agent（语言智能体）

> 来源：姚顺雨（OpenAI）访谈

**定义**：Language Agent是基于[[大语言模型]]构建的具备自我决策能力的系统，它能与外界交互（如使用计算器、互联网或调用各种工具），不仅能生成内容，还能操作和互动。

## 核心观点

**可以推理，因为推理才可以泛化**。

> "我发现，我做完ReAct之后一直在思考一个问题——为什么我可以一下子去玩一个新的游戏，但现在这些系统或AI需要几十万步甚至几百万步训练，才能完成类似的事？
> 
> 我发现，是因为我可以思考。我看到一个全新的环境，会想：这个灯是黑的，那可能有危险，基于常识可能有怪兽；我现在最重要的是点亮灯。基于之前的上下文（Context），灯在我后面，那我应该先向后走。
> 
> 如果没有这样的思考能力，而是直接从复杂语言去预测'我要往后走'，就很难——没有推理做不到。"
> — 姚顺雨

- 语言模型提供了足够强的先验（prior），这个先验让Agent可以推理
- 推理可以在不同环境间泛化
- 核心是推理能力，推理才能带来泛化

## 与BERT的本质区别

| 特性 | BERT | Language Agent |
|------|------|----------------|
| 模型类型 | 判别式模型 | 生成式模型 |
| 能力 | 选择能力 | 自由产生新动作的能力 |
| 行为空间 | 封闭集合 | **open-ended（开放）** |
| 泛化能力 | 有限 | 强 |

> "世界的本质就是，你的行为空间是open-ended的，这种在开放空间决策的能力BERT永远做不到。我发现这个之后，就再也没用过BERT。"
> — 姚顺雨

## 三波Agent演进

### 第一波：符号主义AI
- 基于规则的AI，可以造出早期智能体
- 瓶颈：无法覆盖所有可能发生的情况

### 第二波：深度强化学习
- AlphaGo、DeepMind玩游戏
- 问题：每做一个新环境都要做很多Environment-Specific工程
- 没法泛化：在一个环境学到的东西没办法迁移到另一个环境

### 第三波：大语言模型Agent
- 核心区别：可以推理
- 推理可以在不同环境间泛化
- 从玩游戏到编程、互联网、数字环境

## 核心能力维度

- **Context/Memory处理能力**：实现[[终身学习]]的基础
  - 最看重的能力，因为是Lifelong Learning的基础
- **工具调用能力**（Tool Use）
- **指令遵循能力**（Instruction Following）
- **长上下文推理能力**

## 重要研究方向

- [[Memory|记忆]]：
  - 短期记忆：工作记忆
  - 长期记忆：跨会话持久化
- **Intrinsic Reward**（内生奖励机制）：
  - 创新者最重要能力，如证明黎曼猜想前没有外部奖励
- **Multi-Agent**（[[多智能体系统]]）：
  - L5组织者的核心

## OpenAI能力分级中的位置

| Level | 名称 | 说明 |
|-------|------|------|
| L1 | 聊天机器人 | Chatbot |
| L2 | 推理者 | Reasoner |
| L3 | **智能体** | Agent ← Language Agent |
| L4 | 创新者 | Innovator |
| L5 | 组织者 | Organization |

> "Agent建立在Chatbot和Reasoner能力之上。最关键的两个方向：一个是让它拥有自己的reward，能自己探索；另一个是Multi-Agent，让它们之间能形成组织结构。"
