## 🚀 快速开始

### 1. 打开 Obsidian

在 Obsidian 中打开此仓库，确保已启用以下核心插件：
- **图谱 (Graph)**：查看知识关联
- **模板 (Templates)**：使用笔记模板
- **日记 (Daily Notes)**：记录每日学习
- **标签页 (Tag Pane)**：管理标签

### 2. 创建笔记

#### 方式一：从模板创建
1. 按 `Ctrl/Cmd + P` 打开命令面板
2. 输入 `模板`，选择 `插入模板`
3. 选择对应的模板（学习笔记 / 概念卡片 / 项目 / 每日笔记）

#### 方式二：手动创建
1. 在对应文件夹中新建 Markdown 文件
2. 复制模板内容并填写

### 3. 建立知识链接

在笔记中使用 `[[笔记名]]` 创建双向链接：

```markdown
# 智能体（Agent）

智能体是...（定义）

## 关联知识

- [[大语言模型]]
- [[ReAct 范式]]
- [[智能旅行助手项目]]
```

### 4. 查看知识图谱

1. 按 `Ctrl/Cmd + G` 打开图谱视图
2. 查看笔记之间的关联
3. 发现知识盲区，补充链接

## 🏷️ 标签使用

### 标签层级

| 层级 | 标签类型 | 示例 |
|------|----------|------|
| 第一层 | 知识类型 | `#概念` `#原理` `#方法` `#工具` `#实践` `#案例` |
| 第二层 | 学习阶段 | `#学习笔记` `#待整理` `#概念卡片` `#永久笔记` `#项目` |
| 第三层 | 主题领域 | `#智能体` `#LLM` `#记忆系统` `#多智能体` `#评估` `#训练` |

### 标签使用规则

1. **每篇笔记至少一个标签**：确保笔记可检索
2. **优先使用具体标签**：如 `#ReAct` 而非仅 `#智能体`
3. **每篇笔记 3-5 个标签**：避免过度标签

## 📝 学习工作流

### 每日学习流程

```
1. 打开 Obsidian
2. 查看今日学习目标和计划
3. 阅读教材章节
4. 创建学习笔记（使用"学习笔记模板"）
5. 提取核心概念，创建概念卡片（使用"概念卡片模板"）
6. 完成章节练习，记录疑问
7. 每日复盘（使用"每日笔记模板"）
```

### 项目实践流程

```
1. 确定项目目标
2. 创建项目笔记（使用"项目模板"）
3. 实现过程中记录问题和解决方案
4. 项目完成后进行复盘总结
5. 更新相关概念卡片
```

### 知识整理流程

```
1. 定期回顾"00-Inbox"中的临时笔记
2. 将成熟的笔记移动到"04-Permanent"
3. 精炼概念，创建概念卡片
4. 建立知识之间的链接
5. 更新 MOC 文件
```

## 💡 使用技巧

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + O` | 快速打开文件 |
| `Ctrl/Cmd + G` | 打开图谱视图 |
| `Ctrl/Cmd + Shift + F` | 全局搜索 |
| `[[` | 插入链接 |
| `#` | 插入标签 |

### 搜索技巧

```
#智能体            → 搜索包含该标签的所有笔记
#学习笔记 #待整理   → 搜索同时包含两个标签的笔记
"Thought-Action"   → 搜索精确短语
file:第一章        → 搜索文件名包含"第一章"的笔记
```
